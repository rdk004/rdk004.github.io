## Featured Project — SPARK

SPARK is a stability-optimised framework for reconstructing higher-order transcriptomic pathway architectures using multimodal filtering, graph-based organisation, Leiden community detection, and stability optimisation.

### Repository

https://github.com/rdk004/SPARK

### Core features

- Gaussian mixture model-based pathway filtering
- ΔBIC-driven multimodality detection
- Graph-based pathway organisation
- Stability-optimised Leiden community detection
- Module eigengene reconstruction
- Transcriptomic program architecture modelling
